
## Comando para gerar o projeto
```bash
mvn archetype:generate -DgroupId=imc
   -DartifactId=CalculadorImc
   -DarchetypeArtifactId=maven-archetype-quickstart
   -DinteractiveMode=false
```

## Comando para compilar
```bash 
mvn package
```
